DeLorean Project Gonna go back in time

Project Description

The DeLorean is an automobile-based time travel device capable of going to any point in the past or the future. It is the first device to have successfully traveled through time. The project required updating the software for the on board controls including the destination time display and speedometer as both had original software from 1985. Software updates were also needed when an updated fuel system was added as well as when a conversion kit allowing flight was added.

Project Details

  • Software updated for on-board controls and displays
  • Software required for new fuel system and to allow flight
  • Worked with engineers to ensure compatibility of software with flux capicitor
  • Went back in time to ensure total functionality of software
